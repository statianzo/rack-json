# Rack::Json

Rack middleware for parsing JSON requests.

## Installation

Add this line to your application's Gemfile:

    gem 'rack-json'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rack-json

## Usage

Add it to your config.ru

```ruby
use Rack::JSON
end
